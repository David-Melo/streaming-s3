Streaming-S3

A light-weight S3 upload streaming module for NodeJS.

Features

  • Concurrent part uploading
  • Uses official AWS SDK
  • Simple interface
  • Highly robust
  • Impressively fast
  • Good usage of streams and error handling

Installation

$ npm install streaming-s3

Example Usage

var Streaming-S3 = require('streaming-s3')
